# Rational Game Design: Skill Progression

## Skill Levels

### Reading
| Level | Description                                           |
| ----- | ----------------------------------------------------- |
| R1    | Explicit — short sentences, key info stated directly  |
| R2    | Contextual — info embedded in dialogue or description |
| R3    | Inferential — must combine info from multiple sources |

### Math
| Level | Description                                                   |
| ----- | ------------------------------------------------------------- |
| M1    | Direct — single operation, all numbers given                  |
| M2    | Derived — two operations, one value must be calculated        |
| M3    | Multi-step — chain of operations, intermediate results needed |
| M4    | Modeled — must construct the equation from a word problem     |

### Logical Thinking
| Level | Description                                                              |
| ----- | ------------------------------------------------------------------------ |
| L1    | Single-step world puzzle — one action, obvious goal (push block to reach door) |
| L2    | Multi-step puzzle — sequence matters, intermediate steps needed          |
| L3    | Systemic puzzle — multiple moving parts, must reason about interactions  |

## Design Notes

- **Act I** — tutorial act: I.1 introduces reading/dialogue, I.2 introduces math, I.3 introduces logic puzzles. All three base mechanics established before Act II.
- **I.3 Wizard School Trials** — L1 world puzzle to reach the wizards, then spark test framed as failure. Puzzle must be easy enough to solve; the "no magic" verdict feels arbitrary, not earned.
- **II.5 The Mysterious Letter** — first Reading isolation. R2 delivered via a puzzle object (the letter) rather than dialogue — keeps it concrete for the 8–12 audience.
- **Logic first introduced at I.3**, returns at III.1 with new context (Guild tools), escalates to L2 at III.4. The I.3 introduction is low-stakes; Act III is where it becomes a real mechanic.
- **III.3 The Ghost of Beaverford** — first R3 challenge. Villagers piecing together who is fixing things from multiple observations is naturally inferential. Low narrative stakes make it a safe pre-training step before R3 appears at the climax in IV.3 and IV.5.
- **M4 appears only in Act V** — modeled problems are the culminating math skill, payoff for the full arc.
- **Background reading in Act V stays at R2** — drops from the R3 peak in Act IV. Player is mastering math and logic; no need to maintain high reading load through the resolution.
